# Standalone HTML/CSS Site

This is a standalone version of the educational resource tracker with all 8 pages. It requires no backend and works entirely in the browser.

## Files

- `index.html` - Step 1: Set Up Your GitHub Account
- `page2.html` - Step 2: Microsoft Learn & GitHub Integration
- `page3.html` - Step 3: Microsoft Learn GitHub & Open Source Plan
- `page4.html` - Step 4: Introduction to Git & GitHub
- `page5.html` - Step 5: GitHub Markdown & README
- `page6.html` - Step 6: GitHub Issues & Pull Requests
- `page7.html` - Step 7: Starting Your Own Open Source Project
- `page8.html` - Step 8: Program Completion
- `styles.css` - Shared stylesheet with dark theme

## Usage

Simply open `index.html` in a web browser. All pages are linked together with navigation buttons.

## Features

- No backend required - works offline
- Dark theme matching the original design
- Progress indicator showing current step
- Microsoft Learn Plan button on all pages
- Embedded YouTube videos
- Responsive design for mobile and desktop
- Local storage for form data (GitHub username, email)

## Notes

- Form submissions are stored in browser localStorage for demo purposes
- All external links open in new tabs
- Videos are embedded from YouTube
- The Microsoft Learn Plan link is available on all pages
